# Team Avatar Images

Place team member avatar images in this directory.

## File Naming Convention

- Use lowercase names
- Format: `{firstname}.jpg` or `{firstname}.png`
- Example: `vike.jpg`, `alex.jpg`

## Supported Formats

- JPG/JPEG
- PNG
- WebP

## Image Specifications

- Recommended size: 400x400 pixels (square)
- Minimum size: 200x200 pixels
- Maximum file size: 2MB
- Format: Square aspect ratio (1:1)
